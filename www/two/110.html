110.TLEUNAK KUAT AW

Key-G Missa Hlan Sakding Hla

1.Sualthim sung ah a tang mi thlarau an um, Tleunak kha, Run kuat aw, Runnak a duh na in a ral mi an um, Tleunak kha, Run kuat aw.

CHO:   Maw Pathian, sualnak thlah sak dingin, Khua zakip mi kha ko aw, Na hmai ah kun cio ko hai seh law, Kumkhua nang thangṭhat dingah.

2.Na thuṭha sim tu tla ih dilnak bangin, Run ngai in bawm cio aw, Sual hremnak sung in an luat theinak dingin, Tleunak kha, run kuat aw.

3.Bawi Jesuh duhnak mi hmuah hnen ah thleng seh, Tleu nak kha, run kuat aw, Bawi Khrih thinlung bangtukin nei cio hai seh, Tleunak kha, run kuat aw.
