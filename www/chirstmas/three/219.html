219.A ṬANGSUNG AH KAN CAWL

Key-A Hnemnak Hla

1.Hi khual tlawnnak ah thinphan a um lo, Kumkhua in a ṭang sungah kan um, Maw a sunglawi mi daihnak Pathian cu, Kumkhua in a ṭang sung ah kan um.

CHO:   Him ten kan um, Phannak sung in him ten kan um, Him ten kan um, Kumkhua in a ṭang sungah kan um.

2.Hi khual tlawnnak ah thinphan a um lo, Kumkhua in a ṭang sungah kan um, Nitin ten kan fehnak a tleu sinsin, Kumkhua in a ṭang sungah kan um.

3.Ṭihnungza a um lo, phannak a um lo, Kumkhua in a ṭang sung ah kan um, Bawipa naih tikah harnak ka nei lo, Kumkhua in a ṭang sungah kan um.
