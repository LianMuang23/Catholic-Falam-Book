218.NA THEI DAH MAW

Key-Bb Hnemnak Hla

1.Na thei maw? Lungawinak hla thar aw cu, Sui khua pi'n a ring asi, Tha tho ten kan ṭuan ding, Kan caan ṭha cu fangtel kan khawm ding asi.

CHO:   Fangcang kan khawm ding, Bawipa ra tikah, An mai hmai tha cu kan hmu leh ding, Fangtel kan khawm ding, Bawipa rat tik ah, A inn ah kan um khawm ding.

2.In run zoh sal ih sui khuapi ih sin, Kan harsatnak kha in hmu, A thinglamtah phur a zuam tu pawl in, Kan co ding sui lukhuh cu.

3.Sui khua sung in hla mawi an sak tikah, Ka lung in khoh ter asi, Ni ling lak ih harsat kan tuar hman ah, Cuih hla in in nuam ter asi.

4.Sui khua an hla sak aw ring mawi te cu, Lungawinak pangpar hmuan in, Zumnak ruangah kan thleng ding cuih ramah, Thleng ding kan si rei lo in.
