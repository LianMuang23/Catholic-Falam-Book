THLACAMNAK A PHUNPHUN

Pawlpi Hrangih Thlacamnak

A sunglawi bik Pathian, na Pawlpi cu ralkut sung le sualnak ihsin humhim hram awla, ṭha zet ih na duhnak kan tuah theinak dingin na Pawlpi ih thlacamnak aw hi run ngai hram aw tiah Nangmah le Thiang Thlarau thawn kumkhua hmunkhat ih aum mi kan Bawi Jesuh Christu rinsan in kan lo dil asi. Amen.

Pope Bawipa Hrang Ih Thlacamnak

Zumtu pawl uktu le kilkawitu aw Pa Pathian, na Pawlpi kilkhawitu dingih na remruah mi kan Pope....... cu na duhdawtnak thawn run kilkhawi aw. A thuthang ṭha simnak le a ziaza in, na pek mi na tuuruun tla kumkhua nunnuamnak a thlen ter theinak ding in, run zaangfah hram aw tiah kan Bawi Jesuh Christu rinsan in kan lo dil asi. Amen.

Puithiam Pawl Hrangih Thlacamnak

Maw Bawipa, puithiam pawl cu nangmaih puithiam an si ruangah run kilkhawi hram aw. An nunnak tla cu na hnen ah an lo sum asi. Leitlun ih um mi an si ruangah, lei tlun ih nuamnak le ṭhatlonakihsin run humhim hram aw. Thlarau hrang hna an ṭuan caan ih thinnaunak, riahsiatnak le ninghannak an ton caan ah hnangamnak le thinnomnak run pe hram aw. Aw Bawipa, nangmah siar lo midang rinsan ding an nei lo ruangah run kilkhawi hram aw. Nitin te in na hnen ah thawinak a tuahtu an si bangtukin, an sualnak, an aw kam le an tuahnak tla cu run thianghlim ter hram aw. Amen.

Thianghlim Sacrament Upatnak Thlacamnak

Aw Bawipa Jesuh, biakṭheng par ih Thianghlim bik Sacrament sungah Pathian sinak le minung sinak thawn a ngaingai in na um ti hnget zet in kan zum asi. Na hmai ah upatnak thawn kun in kan lo biak asi. Bawipa nangmah hnen lawng ah lawmnak, thangṭhatnak le duhdawtnak cat loin uh seh law.

Aw Jesuh, kanmah tla in zaangfah tuk ruangah sun le zan catloin Sacramentu sungah na um ringring asi. Na hnen ah ra dingin in ko ih na hnen thleng mi hmuahhmuah cu thluasuah in pek asi. Nangmaih zaangfahnak le in duhdawtnak cu ziangtin ka lo rul thei ding.

Na thisen na taksa le na neih mi hmuahhmuah cu kan thlarau rawl ah in pek ṭheh asi. Thinglamtah ah na thisen luang ṭheh tiang in le na ruangpumpi in pek hlei ah leitlun in na tan ta mi na Nu tiangin kan Nu ah na can ter asi. Hi tluk tiangih in duhdawtnak cu ziangtin kan lo rul thei ding?

Tutiang na duhdawtnak rulkir lo ih kan um ruangah kan sual tuk asi. Tuih sin thok in Bawipa, kan lo duh ve thlang asi. Ziang tlak lo kan lo sum mi kan thinlung hi na duhdawtnak mei alh thawn run kang ter aw.

Kan ruangpum, kan thlarau le kan neih mi hmuahhmuah cu Nu Maria kut sungin kan lo sum asi.Na ṭongkam na thupek mi hmuahhmuah thlun dingin kan zuam thlang ding aw. Kan thinlung ngaingai ih kan lo duh theinak dingah in bawm, in zaangfah hram aw. Sualcawh kuan tuar rero lai mi thlarau pawl par ah a hleice in rak zaangfah awla kumkhua nunnuamnak run pe aw. Misual pawl khal sualsirnak thinlung run pek awla run zaangfah hram aw. Amen.

Nu le Pa Hrangih Thlacamnak

Lei le van sersiamtu Pa Pathian milai sersiamnak ah minung tla na tel ter ih an fate tla cawm dingin le zirh dingin na remruat asi. Ziang hmuahhmuah ṭuanvo neitu an si bangtukin an ṭul mi thluasuah run pek hram aw. A ṭulsam mi pawl an ṭuansuak theinak dingin, tleunak run pek hram aw., thanaunak an ton caan ah thazang run pek hram aw. Duhdawt awknak an neih sinsin theinak dingin run bawm hram aw. Cuih an duhdawtnak in harsatnak phunkim tuar theinak run pe aw. Na duhnak bangtukin fate tla ṭhaten kilkhawi theitu nu le pa an si theinak dingin thluasuah run pek hram aw tiah na Fapa Jesuh hmin rinsan in kan lo dil asi. Amen.

Meikang Ramte Ummi Pawl Hrangih Thlacamnak

Aw Bawipa, kumkhua in zaangfahnak neitu na si bangtukin meikang ramte ih aum mi thlarau pawl tla run kilkhawi hram aw. Anmah cu na duhdawtnak in run zaangfah hram awla, an sualnak tla run ngaithiam hram aw.

An sualnak tla cu kholhfai awla nangmah lo tong thei dingin run bawm aw. Rei lo te ih vancungram nunnuamnak an luh theinak dingin run zaangfah aw tiah kan lo dil asi. Amen.

Thih Thianghlim Ih Thih Theinak Hrang Thlacamnak

Minung hmuahhmuah thi dingin rem ruat hman sehla an thih ding caan phuang lo in a umtu Pathian Bawipa, hi leitlun ah ding zet le fel zet ih kan um theinak dingin le thianghlim te ih kan thih theinak dingin in bawm hram aw tiah nangmah le Thiang Thlarau thawn kumkhua in a um tu kan Bawipa Jesuh Christu ih zaangfahnak rinsan in kan lo dil asi. Amen.

Remnak le Daihnak Hrang Thlacamnak

A thianghlim mi saduhthahnak siseh, a hmaan mi ruahsannak siseh, a ding mi tuahnak siseh in petu Pathian Bawipa, na duhnak vekin kumkhua kan um ringring theinak dingin le kan thinlung sungih aum mi ralṭihnak cu run la hlo awla na zaangfahnak ruangin kan san cu dai te ih aum theinak dingin nangmah le Thiang Thlarau thawn hmunkhat in kumkhua aum tu kan Bawipa Jesuh Christu rinsan in leitlun ih a pek thei lo mi remnak le daihnak cu in pe hram aw tiah zaangfahnak kan lo dil asi. Amen.

Rawl Ei Hlan Thlacamnak

Aw Pathian Bawipa, nangmaih laksawng asi mi hi rawl par ah thluasuah run pek awla, kan in ding mi hi kan ruangpumpi le kan thlarau damnak le cahnak ah run can ter dingin kan Bawipa Christu rinsan in kan lo dil asi. Amen.

Rawl Eihnu Thlacamnak

Azate ih thiamtu le kumkhua ih nungtu Bawipa Pathian, na kut sungih kan ngah mi na zaangfah,ak tla ruangah lungawi thu kan lo sim asi. Amen.

Ruahti Dilnak Thlacamnak

Ziang hmuahhmuah sersiamtu Bawipa Pathian, hi leitlun ih kan hman mi hmuahhmuah le kan nunnak tla cu nangmah in, in zaangfah mi laksawng pawl an si. A tuih kan nunnak hrangih a ṭul zetmi ruahti ṭha in pek dingin luat tikah hi leitlun ih kan hnaṭuannak khi thinnuam ten kan ṭuan thei dingih nehhnu san ih nunnak thu khal kan ruat thei ding asi. Kan sualnak ruangah na thluasuahnak khi co tlaktu kan si lo naan, thleidannak nei lo in zaangfah thei ringring tu na si bangtukin a tu ih kan ṭulmi ruahti ṭha in run pek hram aw tiah na Fapa Bawipa Jesuh Christu rinsan in kan lo dil asi. Amen.
