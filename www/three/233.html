233.VANCUNGKHUA KAN THLENG TIKAH

Key-C Hnemnak Hla

1.Jesuh duhnak mangbangza thu, Thangṭhatnak hla kan sak ding, A tleu a nuam kan um nak ding, Inn ṭha rem cia in a um.

CHO:   Vancung khua kan thlen tikah, A sunglawi nuamnak lawmnak kan hmu ding, Jesuh kan hmu tikah, Nehnak hla ring pin kan sak ding.

2.Vancung lam kha kan feh lai ah, Ninghan phun kim kan tuar ding, Kan khual tlawn nak a net tikah, Phannak hmuah in a hlo ding.

3.Ni tin te in kan ṭuan mi khi, Zuamnak thawn ṭuan cio uh si, Zonzaihnak kan tuar mi hmuah in, Pathian in a sam leh ding.

4.Laksawng ṭha hmai ah ret cia mi, A rei hlan ah kan co ding, Lungvar sangkaa a ong tikah, Van inn sung ah kan lut ding.
